# Orehum-bot
- Наипростейший бот на жаве для просмотра персонажей, плейтайма и т.д.
- Утечка памяти в подарок.
# Запуск и сборка
- Делаем файл `.env` куда вписываем конфиг по примеру из файла .env.template
- С докером
- `git pull`
- `./update.sh`
- `./start.sh`
- Без докера
- `git pull`
- `./gradlew jar`
- `java -jar build/libs/big-bob-1.0-SNAPSHOT.jar`
